달기지를 찾는 외계 생명체들. 각 종족은 고유한 특수 능력을 가지고 있으며, 주인공은 이 능력들을 탐사와 구조에 활용한다.

### 종족 목록

#### 크리쉬 (Krish) — 결정 육성자

| 항목 | 내용 |
|------|------|
| 외형 | 인간형, 피부가 얇은 결정질, 키 150cm 내외 |
| 특수 능력 | 광물의 결정 구조를 읽고 성장시킬 수 있음 |
| 활용 | 기지 구조물 보수, 숨겨진 통로 탐지, 유물 분석 |
| 성격 | 조용하고 관찰력이 뛰어남. 말수 적지만 신뢰할 수 있음. |
| 방문 이유 | 달기지 깊은 곳의 희귀 광물 연구 |

#### 발토 (Balto) — 중력 감응자

| 항목 | 내용 |
|------|------|
| 외형 | 4족 보행에 가까운 유연한 체형, 2m 장신 |
| 특수 능력 | 주변 중력을 감지하고 국소적으로 조정 가능 |
| 활용 | 붕괴 위험 구역 안전 진단, 무거운 짐 운반, 사고 구조 |
| 성격 | 호탕하고 붙임성 좋음. 주점에서 가장 즐거워함. |
| 방문 이유 | 지친 몸을 달기지에서 쉬고 싶어서 |

#### 루므 (Lume) — 기억 공명자

| 항목 | 내용 |
|------|------|
| 외형 | 반투명 유영체. 평소엔 작은 구체 형태로 머무름 |
| 특수 능력 | 사물에 남은 감정의 기억(노이즈)을 읽고 정화 |
| 활용 | 유물 폭주 원인 분석, 정신 오염자의 기억 정리, 과거 사건 재구성 |
| 성격 | 조심스럽고 섬세. 타인의 감정에 쉽게 영향받음. |
| 방문 이유 | 달기지의 강력한 정신 에너지에 끌려 방문 |

#### 그 외 방문 종족

- **철기 기술자족:** 기계·드론 정비에 능숙. 쉘터 설비 업그레이드에 큰 도움.
- **식물 공생족:** 달기지 농장 확장에 협력. 새로운 재배 기술 전수.
- **기록 보관자족:** 언어·문자 해독에 능함. 고대 유물의 비문을 해석.

### 능력 활용 원칙

1. 각 종족의 능력은 강제가 아닌 **협력**을 통해 발휘된다.
2. 주인공은 이들의 능력을 탐사와 구조에 적극 활용한다.
3. 밤 주점에서 이들은 각자의 문화와 이야기를 나누며 자연스러운 유대를 형성한다.
4. 위기 상황에서 각자의 특기가 빛을 발하는 순간이 카타르시스를 만든다.

### 핵심 정서
- 다양한 생명체가 공존하는 우주의 풍경
- 각자의 능력이 모여 하나의 팀이 되는 감동
- "달라서 좋은 거야"
